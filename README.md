# CMSR — Village Development & School Empowerment Platform

> "Our Village – Our Responsibility – Our Development"

## About

CMSR is a nationwide digital platform empowering villages and schools across India through donations, volunteer programs, and community-driven initiatives.

## Prerequisites

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file:

```
VITE_BASE44_APP_ID=your_app_id
VITE_BASE44_APP_BASE_URL=your_backend_url
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- **Frontend:** React 18 + Vite
- **Styling:** TailwindCSS + shadcn/ui
- **Animations:** Framer Motion
- **Charts:** Recharts
- **Icons:** Lucide React
